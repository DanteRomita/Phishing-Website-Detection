# Phishing-Website-Detection
A graduate-level university project for detecting phishing websites using machine learning. Here, 5 classical models are compared with 5 gradient boosting models to determine which one performs best. View my attached paper and presentation PDF for additional information!

# Instructions To Run The Project
The following steps provide the easiest way to run the project:

1. Download the project files and run the `.ipynb` file in Visual Studio Code
2. Ensure that Python is installed on your system.
3. Select `Run All`, select a Python installation to run all cells. You will be prompted to install extensions and kernels. Agree to all of them.
4. Wait for all cells to complete and view the results!
