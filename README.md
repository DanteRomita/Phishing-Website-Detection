# Phishing-Website-Detection
A graduate-level university project for detecting phishing websites using machine learning. Here, 5 classical models are compared with 5 gradient boosting models to determine which one performs best. View my [academic paper](https://dan-a-rama.neocities.org/project-pdfs/Phishing%20Website%20Detection%20Using%20Gradient%20Boosting%20Models%20-%20A%20Breadth-focused%20Comparative%20Analysis.pdf) and [presentation PDF](https://dan-a-rama.neocities.org/project-pdfs/Phishing%20Website%20Detection%20Using%20Gradient%20Boosting%20Models%20-%20A%20Breadth-focused%20Comparative%20Analysis%20%5BPresentation%5D.pdf) for additional information!

# Instructions To Run The Project
The following steps provide the easiest way to run the project:

1. Download the project files and run the `.ipynb` file in Visual Studio Code
2. Ensure that Python is installed on your system.
3. Select `Run All`, select a Python installation to run all cells. You will be prompted to install extensions and kernels. Accept all of them.
4. Wait for all cells to complete and view the results!
